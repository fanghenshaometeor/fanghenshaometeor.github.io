研一小结

Posted on 2019-06-16 | Views:

研一就要结束了，写个小结吧。
得之偶然间，失之亦草草。
顾我行囊中，何物随人老。

严严谨谨地总结一下，不搞一些抒情的东西。

¶课程

硕士研究生需要第一年里修满所有的学分。参考了师兄的建议和自己的考量，选了多门数学课。

第一学期：

  • 政治课：自然辩证法概论、中国特色社会主义理论和研究
  • 专业课：智能信息处理理论及应用、数字图像处理
  • 数学课：矩阵理论、基础数理统计、运筹与优化

第二学期：

  • 公共课：学术英语、学术报告会
  • 专业课：计算机视觉、专用控制技术
  • 数学课：应用泛函分析

政治课和英语课没啥好说的。学术报告会听满八次报告填个表交上去即可，面子工程。

专业课的话，智能信息处理理论及应用是老板的课，只去过两次，大作业也没做，第二学期开学交了一份心得体会权作大作业报告。数字图像处理课让我非常失望，教的太水了，课件杂乱无章、不成体系、反复重复一部分内容（人类的本质？），课上也没听，难以想象某老师是怎么讲下来的。计算机视觉全英课，当然上课也没听，在做自己的事情，中间有一个small project，最后有一份小组大作业，写了proposal和报告、做了presentation，把在做的项目整理了一下作为大作业成果了。专用控制技术写份项目报告老板签字给成绩交掉即可。

至于数学课，矩阵理论和基础数理统计的两位老师还是教的非常好的，非常满意，尤其满意矩阵理论。运筹与优化和应用泛函分析则较为失望，尤其对运筹与优化非常失望。应用泛函分析讲到泛函的定义的时候，已经是第二学期的倒数第二节课了。

现在仍然对数学学习存在一些困惑，搞科研必然离不开数学，可是数学课上学到的数学又和实际的科研工作存在距离。有一种隐隐约约的感觉是：只需要对一些基本概念和体系有所了解即可，然后在不断地阅读科研论文、推导公式证明的过程中巩固并拓宽自己的数学知识。当然也不可否认，数学课学习对于加强基本概念体系理解的作用。

¶项目

一年来做了两个项目，分别记作A和B。

在项目A里面，完全作为打下手的工作，把MATLAB代码转化成C/C++代码。从暑假就开始做这个工作了，一直做到了第一学期末。后来由于项目B来了，便慢慢从项目A退出了，只偶尔做一些活。项目A是横向的项目，老板和外面的公司合作，我个人对对方公司和这个项目持保留意见。

项目B同样是横向项目，和行业世界五百强公司合作，过程磕磕绊绊，不过总体体验还是很好的。公司给了几个实习岗位，拿到了其中一个。每周只用去一次公司。体验到了外企工作的闲适，朝九晚五，一周双休。真的舒服！

从项目B中，深刻地理解了工业界需求和学术界研究的差异。在CV领域，论文里面做出来的东西，即便它claim的效果再好，如果不能落地到实际产品中，便是没有用的。而在这落地到实际产品的过程中，又会遇到很多工程上的问题。譬如，模型前向推理非常快，但是工程上受限的瓶颈在于显存和内存之间数据的搬运。

¶Research

第一学期课程比较多，也没有太多的时间去搞科研。不过从头到尾参与了实验室里做Hashing的师兄的组会，黄老师也来旁听指导，这一组会主要是为了引导新生入门科研，如果有愿意做hashing这个方向，便可跟着师兄做。不过我对这一方面并没有太大的兴趣，所以之后也没有继续了。

等到第一学期期末考试结束，便开始一点点地接触科研了。

最开始看了一些大规模 Kernel Methods 加速方向的一些论文，后来在黄老师的指导下，从其中的一个方向 Random Fourier Features 出发，结合了 Generative Model，在 Kernel Learning 的方向上做了一些新工作。这样做了一学期，其间曲折，现在回头看来也是不值一提。最后也确确实实产出了一些东西，投出了自己的第一篇文章。写文章的过程也是一言难尽……

终归是自己在科研方向的第一次尝试吧，接下来要加强自己的数学功底，在保证生存的基础上，追求做一些更加solid的工作。

¶其它

第一学期一直在打农药，一度打到了星耀三还是四来着，寒假回家也在玩，不过找不到家里的亲戚一起玩。他们要么在玩抖音，要么在吃鸡。

第二学期倒是弃了农药，开始玩单机了。先是通关了隐形守护者的全结局，很受触动。一周目的时候选择下了火车，后面被高源玩了，杀了第二号，感觉不对劲，反复尝试改选择也改变不了事实，果断弃了，选择留在火车。于是顺利地走到红色芳华线结局。二周目扶桑线，三周目先后丧钟为谁而鸣和美丽新世界。懦夫畏死终须死，志士求仁几得仁。向那个年代坚持信仰的地下工作者们致敬。

刺客信条二代全收集全支线通关，印象比较深刻的几个场景，开篇Ezio和哥哥的月下谈心，“看啊，弟弟，我们的生活多么美好！”，是悲剧降临前的最后一点温暖；Ezio在威尼斯的高塔上加入组织、吟诵信条：万物皆虚、万事皆允；行于黑暗、侍奉光明。Nothing is true, Everything is permitted。本来想把Ezio三部曲全部打完再玩别的游戏的，不过到后来强迫症对收集任务不厌其烦、游戏模式单一、主线也不够吸引人，没啥继续玩这一类型游戏的欲望了。

By the way，意大利语真的很有韵味。

其间断断续续地玩了一段时间的古墓丽影，Lara小姐姐真好看。

最近在玩黑暗之魂三，受虐游戏。灰烬审判者古达真新手劝退怪，古老师教会了我锁定敌人、教会了我犹豫就会败北、教会了我观察敌人的行动攻击模式。不得不说黑魂三的设计平衡性真的很好，总是会觉得自己操作再好一点就可以赢，而且也确实是这样，不至于让人绝望，也不至于让人无聊，会有一种自己真的在变强的感觉（大雾）。玩惯了刺客信条的无双，再玩一玩这个，还是蛮有趣的。
